Ringzzle is a free color rings puzzle game for quick browser play. No install, account, or email is required.

[Play Ringzzle](/play/)

## Board and tray

Ringzzle uses a 3x3 board and a bottom tray with three pieces. Each tray piece is one colored ring.

Each board cell has three ring slots:

- small
- medium
- large

A cell can hold one ring of each size. Different sizes can sit in the same cell, but the same size cannot be placed twice in one cell.

## Clearing lines

Clear a row, column, or diagonal by matching the same color across all three cells. Ring size does not need to match for a line clear.

When a line clears, the matching-color rings in that line disappear. Other colors in the same cells stay on the board.

## Color Burst

If one cell contains small, medium, and large rings of the same color, that cell triggers a Color Burst. Color Burst clears all rings of that color from the board.

## Score and leaderboard

Valid placements add score. Line clears, combo clears, and Color Burst add bonus score.

Ringzzle stores your local high score in your browser. After game over, you can optionally submit a nickname-only score to the anonymous leaderboard. There is no login, no email, and no fake ranking data.
